// App Name

- AppHub — App Store Platform

// Description

 - AppHub is a modern and responsive app store platform where users can explore trending applications, view detailed information, and install their favorite apps. It provides a smooth user experience with dynamic routing, live search, and persistent installation tracking using localStorage.

// Features
- Live search (case-insensitive)
- Fully responsive design (mobile, tablet, desktop)
- App details with ratings & downloads
- Dynamic rating visualization
- Install / Uninstall functionality
- Data persistence using localStorage
- My Installations page
- Sort apps by downloads (High-Low / Low-High)
- Page navigation loading animation
- Custom Error & Not Found pages

// Tech Stack
- React.js
- React Router DOM
- Tailwind CSS
- Recharts
- LocalStorage
